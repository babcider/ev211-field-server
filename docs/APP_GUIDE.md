# Android·iOS 앱 사용 안내

EV211은 Android와 iOS에서 같은 흐름으로 동작합니다. 앱 화면 오른쪽 위의 지구본 아이콘에서 표시 언어를 바꿀 수 있습니다.

App Store와 Google Play 링크는 출시 심사 완료 후 여기에 추가됩니다.

- App Store. 출시 준비 중입니다.
- Google Play. 출시 준비 중입니다.

## 로컬서버접속

이 저장소로 설치한 Ubuntu 서버에 접속하는 모드입니다. 단말과 서버가 같은 Wi-Fi에 있어야 합니다.

### 음성 듣기

1. 첫 화면에서 `로컬서버접속`을 선택합니다.
2. 현장 관리자가 알려 준 서버 IP를 입력합니다.
3. `수신 접속`을 누릅니다.
4. 채널 목록에서 원하는 언어나 `Floor(원어)`를 선택합니다.
5. `청취 중` 화면의 수신 바로미터가 움직이는지 확인합니다.

송신이 아직 시작되지 않은 채널은 `송신 대기 중`으로 표시되며, 송신자가 시작하면 자동으로 들립니다. 이어폰이 분리되거나 전화가 오면 청취가 일시정지될 수 있으므로 화면의 `다시 듣기` 또는 `청취 시작`을 누르세요.

### 채널 송신

1. `로컬서버접속`에서 서버 IP를 입력합니다.
2. `송신자이신가요?`를 누르고 현장 송신 비밀번호를 입력합니다.
3. 비어 있는 채널을 누르거나 `채널 개설`로 번호와 언어를 정합니다.
4. 마이크 권한을 허용하고 `송신 시작`을 누릅니다.
5. 마이크 입력 바로미터가 움직이고 상태가 `송신 중`인지 확인합니다.

원어를 동시에 들어야 하면 `Floor(ch-00) 동시 수신`을 켭니다. 스피커로 원어를 들으면서 송신하면 에코가 생길 수 있으므로 이어폰이나 헤드셋을 권장합니다. 송신을 마치면 반드시 `송신 정지`를 누르세요.

### 무전기

1. `로컬서버접속`에서 `무전기(인터컴) 모드`를 선택합니다.
2. 송신 비밀번호와 선택 사항인 표시 이름을 입력합니다.
3. 기존 채널에 입장하거나 `새 채널`을 만듭니다.
4. 큰 `누르고 말하기` 버튼을 누르는 동안 말합니다.
5. 계속 말해야 하면 `연속 송신(핸즈프리)`를 켭니다.

송신 바로미터는 내 마이크 입력을, 수신 바로미터는 상대 음성을 표시합니다. 연속 송신은 주변 소리를 계속 전달하므로 사용을 마치면 바로 끄세요. 전화나 오디오 인터럽션 중에는 마이크가 자동으로 닫힙니다.

### 관리자

`로컬서버접속` 화면 아래의 `관리자`를 누르고 관리자 비밀번호를 입력합니다. 관리자는 채널과 송신 상태, 청취자 수, 실시간 송신 바로미터를 확인하고 채널 삭제와 비밀번호 변경을 수행할 수 있습니다.

서버 PC에서는 `http://localhost:8880/admin.html`, 다른 내부망 PC에서는 `https://서버IP:8443/admin.html`을 엽니다. 관리자 모드에서 관리자·송신자 비밀번호를 각각 변경하고 방송 채널과 무전기 채널의 참가자 수·오디오 바로미터를 함께 볼 수 있습니다. 채널별 `녹음 시작`과 `녹음 종료`를 누르면 서버가 MP3를 만들며, 완료 후 `MP3 다운로드`로 받을 수 있습니다.

송신자 비밀번호를 변경하면 현재 세대가 회전되어 진행 중인 송신과 수신이 모두 끊깁니다. 운영 중에는 충분히 공지한 뒤 변경하세요.

## 폰서버접속

이 모드는 별도 Ubuntu 서버 없이 한 대의 폰이 서버 역할을 합니다. 이 저장소의 프로그램이나 비밀번호를 사용하지 않습니다.

### 한 폰에서 방송하기

1. 모든 폰을 같은 Wi-Fi에 연결합니다.
2. 호스트 폰에서 `폰서버접속` → `이 폰에서 방송하기` → `폰서버 시작`을 누릅니다.
3. 화면의 호스트 IP, 6자리 참가코드 또는 QR을 청취자에게 전달합니다.
4. 마이크 입력 바로미터가 움직이는지 확인합니다.
5. 끝나면 `방송 종료`를 누릅니다.

청취자는 `폰서버접속` → `폰서버 방송 듣기`에서 IP와 참가코드를 입력하거나 QR을 스캔합니다. 수신 오디오 바로미터가 움직이는데 소리가 들리지 않으면 미디어 음량과 Bluetooth 출력 장치를 확인하세요.

### 폰서버 무전기

호스트는 `폰서버접속` → `폰서버 무전기` → `이 폰에서 무전기 시작`을 선택합니다. 참가자는 `폰서버 무전기 참가`에서 호스트 IP와 참가코드를 입력하거나 QR을 스캔합니다.

호스트와 참가자 모두 PTT와 `연속 송신(핸즈프리)`를 사용할 수 있습니다. 호스트 폰을 포함해 최대 8명을 대상으로 하며, 실제 품질은 호스트 폰과 Wi-Fi 성능에 따라 달라집니다.

## 처음 접속할 때의 인증서

로컬 Ubuntu 서버는 자체 서명 인증서를 사용합니다. 앱은 처음 본 인증서를 해당 서버의 인증서로 기억합니다. 같은 IP의 인증서가 나중에 바뀌면 앱이 경고합니다.

현장 관리자가 서버를 재설치하거나 인증서를 재발급한 사실을 확인한 경우에만 `계속 신뢰`를 선택하세요. 확인할 수 없다면 접속을 취소하고 관리자에게 문의하세요.

## 소리가 들리지 않을 때

- 송신·수신 바로미터 중 어느 쪽이 움직이지 않는지 먼저 확인합니다.
- 모든 단말이 서버 또는 호스트 폰과 같은 Wi-Fi인지 확인합니다.
- 공유기의 AP 격리 또는 게스트 네트워크를 사용하지 않았는지 확인합니다.
- Android와 iOS 설정에서 EV211의 마이크 및 로컬 네트워크 권한을 허용합니다.
- Bluetooth 이어폰에 연결돼 소리가 다른 장치로 나가는지 확인합니다.
- 로컬서버 모드는 관리자에게 `sudo ev211ctl doctor` 실행을 요청합니다.
