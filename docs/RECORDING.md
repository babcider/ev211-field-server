# 채널 녹음 안내

관리자 웹은 방송 채널과 무전기 채널을 서버에서 직접 녹음합니다. 녹음 중 관리자 브라우저를 닫아도 서버의 field-api가 LiveKit 오디오를 계속 구독합니다.

## 녹음 시작과 종료

1. PC를 서버와 같은 내부망에 연결합니다.
2. `https://서버IP:8443/admin.html`을 열고 관리자 비밀번호로 접속합니다.
3. 방송 또는 무전기 채널 행에서 `녹음 시작`을 누릅니다.
4. `서버 녹음 파일` 표에 `녹음 중`이 표시되는지 확인합니다.
5. 종료할 때 해당 채널이나 녹음 파일 행의 `녹음 종료`를 누릅니다.
6. MP3 변환이 끝나면 `MP3 다운로드`를 누릅니다.

방송 채널은 해당 `ch-NN` 트랙만 녹음합니다. 무전기 채널은 동시에 말하는 참가자의 오디오를 하나의 모노 MP3로 혼합합니다. 녹음 중 송신자가 잠시 나가도 녹음 세션은 유지되며, 다시 들어온 오디오를 이어서 받습니다.

## 저장 위치와 백업

MP3와 메타데이터는 `ev211_field_api_data` Docker volume의 `/data/recordings`에 저장됩니다. `sudo ev211ctl backup`은 SQLite와 녹음 파일을 함께 백업해야 합니다.

녹음 파일에는 참가자의 음성이 들어 있으므로 개인정보와 현지 법률, 행사 동의 절차를 확인하세요. 다운로드한 파일과 서버 백업은 접근 권한이 제한된 암호화 저장소에 보관하는 것을 권장합니다.

## 용량 계획

송신 원본은 Opus 최대 96kbps 고음질 프리셋을 사용하며, 서버는 이를 48kHz 모노 MP3, 128kbps로 저장합니다. 한 시간당 대략 60MB가 필요합니다. 녹음 시작 전 서버의 여유 공간을 확인하세요.

```bash
df -h
sudo du -sh /var/lib/docker/volumes/ev211_field_api_data/_data/recordings
```

저장된 파일은 자동 삭제되지 않습니다. 필요한 MP3를 다운로드하고 백업한 뒤, 디스크 정리는 서버 관리자가 신중하게 수행해야 합니다.
